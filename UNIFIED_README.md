## 🚀 One-Click Deployment

### Prerequisites

```bash
# Required tools
node --version    # >= 18.0.0
npm --version     # >= 8.0.0  
python3 --version # >= 3.8.0

# Optional: For live deployment (otherwise runs in simulation)
export DEPLOYER_KEY="your-wallet-private-key"
```

